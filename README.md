# weargram
# 概要
洋服が好きな人の為に洋服を記録できるウェブサイトを作成しました
# 使い方
自分のコーディネートを投稿することが出来たり、おすすめユーザーから自分の好きなユーザーを探し、コーディネートを見る事ができます。
気に入ったコーディネートはいいねをする事ができたり、気に入ったユーザーをフォローする事ができます。
